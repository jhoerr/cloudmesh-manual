cloudmesh.group.Group module¶

class cloudmesh.group.Group.Group[source]¶

Bases: object

Groups are used to store the names of services that are part of the group. Members are identified by their name and the kind (such as vm).

The data of a group is managed via a dict. Here is a simple example

cm:

name: test cloud: local kind: group

members: - name: vm-1

kind: vm

  • name: vm-2 kind: vm

  • name: vm-3 kind: vm
